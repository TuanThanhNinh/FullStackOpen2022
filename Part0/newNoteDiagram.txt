    


 user->>+browser: type the new note and click on the submit button
 browser->>+server: POST https://studies.cs.helsinki.fi/exampleapp/new_note
 server->>+browser: responds with HTTP status code 302, 
 browser-->+server:  three HTTP requests: fetching the style sheet (main.css), the JavaScript code (main.js), and the raw data of the notes (data.json).

	 The server creates a new note object and adds it to an array called notes.

 server-->+browser: a response indicating that the note has been successfully created
 	The browser updates the page to display the newly created note along with any other existing notes.
